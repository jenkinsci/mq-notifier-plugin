With this setting, each message gets a routing key based on the message sent.

Values for automatic setting of routing keys:
  • Queue
    • Entered: queue.QUEUED
    • Left: queue.DEQUEUED
  • Run
    • Started: run.STARTED
    • Completed: run.COMPLETED
    • Deleted: run.DELETED
  • Executor
    • Task accepted: executor.TASK_ACCEPTED
    • Task started: executor.TASK_STARTED
    • Task completed: executor.TASK_COMPLETED
